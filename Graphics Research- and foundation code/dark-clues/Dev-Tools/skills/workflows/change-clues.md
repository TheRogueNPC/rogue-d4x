# Workflow: Change Clue Text

## Prerequisites
- Read `modules/IN_OUT_GLOSSARY.md` §1 (Asset Pipeline — JSON Data)
- Read `modules/tracery/tracery.js` — understand the Tracery grammar system
- Read `assets/clues.json` — the current grammar

## How Tracery Works

Clue text is generated by a **Tracery grammar**. A grammar has:
- **Symbols** — named non-terminals (e.g. `#clue#`, `#tarot#`)
- **Rules** — replacement strings for each symbol
- **Modifiers** — text transforms (capitalize, pluralize, etc.)
- **Inline choices** — `{a|b|c}` random selection syntax

Example from `clues.json`:
```json
{
  "clue": [
    "a {revolver|handgun} loaded with silver bullets",
    "the victim's {diary|journal} with pages torn out"
  ]
}
```

When `grammar.flatten("#clue#")` is called, it picks a random rule from the
`clue` symbol, expands any inline `{choices}`, and returns finished text.

## Steps

### 1. Edit the grammar file

Open `assets/clues.json` and modify the rules:

```json
{
  "clue": [
    "your new clue text here",
    "another clue with {choice_a|choice_b} inline options",
    "a clue referencing the #tarot# card"
  ],
  "tarot": [
    "the Fool",
    "the Magician",
    "the Tower"
  ]
}
```

**Rules for good clue text:**
- Use `#symbol#` to reference other symbols
- Use `{option1|option2|option3}` for random choices
- Use `0.3?-{rule}` for weighted probability (30% chance)
- Use `.capitalize` modifier: `#name.capitalize#`

### 2. Add new symbols

Add new symbol arrays to the JSON:

```json
{
  "clue": ["a #object# found in the #location#"],
  "object": ["knife", "locket", "letter"],
  "location": ["attic", "garden", "cellar"]
}
```

### 3. Change the root symbol (optional)

To generate different types of text, change the root symbol in the `Clue`
constructor (`characters.js`):

```js
// Current: this.text = Clue.grammar.flatten("#clue#");
// Change to: this.text = Clue.grammar.flatten("#rumor#");
```

### 4. Add grammar modifiers (optional)

English text modifiers are in `tracery/tracery.js` (`ModsEngBasic`):
- `.capitalize` — "hello" → "Hello"
- `.capitalizeAll` — "hello world" → "Hello World"
- `.a` — "apple" → "an apple"
- `.s` — "clue" → "clues"
- `.ed` — "walk" → "walked"
- `.ing` — "walk" → "walking"
- `.possessive` — "John" → "John's"

Add custom modifiers to the grammar:
```js
Clue.grammar.addModifiers({
    "reverse": function(str) { return str.split("").reverse().join(""); }
});
```

### 5. Test

Start a new game and collect clues. Each clue should show new text.
Check the console for Tracery errors if text shows `((symbol))`.
